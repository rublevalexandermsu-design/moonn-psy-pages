# Moonn Psy Pages

Статический SEO/AEO-контур для Татьяны Мунн.

## Канон

- Основной брендовый сайт остаётся на `moonn.ru` в Tilda.
- Этот репозиторий публикует отдельный поддомен `tatyana-psy.moonn.ru`.
- Источник истины для страниц: `data/site.json`.
- Сборка: `build_site.py`.

## Как работать

```powershell
python build_site.py
```

После сборки сайт появляется в `dist/`.

## Зачем это нужно

- быстро править посадочные страницы через Git;
- держать единый AEO/IEO-слой для людей и агентов;
- не ломать корень `moonn.ru`, который уже живёт на Tilda;
- масштабировать запросы по кластерам без ручной сборки блоков.
